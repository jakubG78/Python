Create a function that converts miles to kilometers. The function should take a single parameter for the distance in miles and return the equivalent distance in kilometers. Use the formula: 1 mile = 1.60934 kilometers. Round the result to two decimal places. Your program should read a single float value as input (miles) and print the converted distance in kilometers.


Sample Input:
5.0

Sample Output:
8.05


Sample Input:
10.5

Sample Output:
16.90



Memory limit: 256 MB
Time limit: 15 seconds
