Zoe is taking an online class and needs to know whether to study, rest, or do her usual activities according to a program-designed schedule. Create a program for Zoe. The program will take an integer from 0 to 6 (both inclusive) representing days of the week from Monday to Sunday respectively. In the case of a weekday, the program should print 'Study!'. If it's Saturday (5), the program should print 'Rest!', and if it's Sunday (6), the program should print 'Leisure!'.


Sample Input:
0

Sample Output:
Study!


Sample Input:
5

Sample Output:
Rest!



Memory limit: 256 MB
Time limit: 15 seconds
